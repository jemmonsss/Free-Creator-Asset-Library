# Asset Licensing Information

This repository uses **two different licenses** for two different things.

Understanding this is important to avoid confusion.

---

# Repository License (MIT)

The repository itself is licensed under the **MIT License**.

This applies to:

- Documentation
- Folder structure
- Repository organization
- Scripts
- Configuration files

The MIT license **does NOT apply to the assets themselves**.

See `LICENSE` for details.

---

# Asset License (CC0)

All assets included in this repository must be released under:

**Creative Commons Zero (CC0)**

CC0 places the asset in the **public domain**.

This means anyone can:

- Use the asset
- Modify the asset
- Use it commercially
- Redistribute it
- Include it in other asset packs or projects

No attribution required.

---

# Why CC0?

The goal of this repository is to provide assets that developers and creators can use **without legal complications**.

Using CC0 ensures:

- Maximum freedom
- No attribution requirements
- No restrictions on use

---

# Contributor Responsibility

By submitting assets, contributors confirm that:

- They created the asset themselves **OR**
- They have legal permission to redistribute it under CC0

If an asset violates licensing rules, it may be removed.
